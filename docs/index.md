## Quick Install

```bash
# 1. Upload the plugin to your WordPress site
# 2. Activate it in Plugins > Installed Plugins
# 3. Go to WooCommerce > LiteQuote to configure
# That's it. No complex setup required.
```

## Requirements

- WordPress 6.6+
- WooCommerce 8.0+
- PHP 8.0+
- An SMTP plugin (FluentSMTP recommended) for email delivery
