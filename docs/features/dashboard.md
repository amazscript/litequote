# Quote Dashboard

The dashboard gives you a complete overview of all quote requests received through your shop.

## Access

Go to **WooCommerce > Quotes** in the admin menu.

A **red badge** shows the number of pending requests that need your attention.

## Quote List

The table shows:

| Column | Description |
|---|---|
| Checkbox | Select for bulk actions |
| Reference | Unique quote ID (e.g., LQ-2026-0042) |
| Customer | Name + company |
| Product | Product name |
| Qty | Requested quantity |
| Date | Submission date |
| Status | Current status with color indicator |
| Actions | View / Reply buttons |

## Statuses

| Status | Color | Meaning |
|---|---|---|
| **Pending** | Orange | New request, waiting for your response |
| **Quoted** | Blue | You sent a quote with a price |
| **Accepted** | Green | Customer accepted your quote |
| **Rejected** | Red | Customer declined or you rejected the request |

## Filtering

Click the status tabs at the top to filter:

- **All** -- every request
- **Pending (X)** -- needs your action
- **Quoted (X)** -- awaiting customer response
- **Accepted (X)** -- won quotes
- **Rejected (X)** -- declined quotes

## Search

Search by customer name, email, product name, or reference number.

## Bulk Actions

1. Check the boxes next to the quotes you want to update
2. Select an action from the dropdown:
   - **Mark as Accepted**
   - **Mark as Rejected**
   - **Delete**
3. Click **Apply**

## Quote Detail Page

Click **View** on any quote to see the full detail:

- **Customer info**: name, company, email (mailto link), phone (tel link)
- **Product info**: name, SKU, variation, quantity, links to view/edit
- **Customer message**: the original request
- **Your quote** (if sent): unit price, quantity, discount, total, validity, notes
- **Actions sidebar**: Send a Quote, Reply by Email, Change Status
