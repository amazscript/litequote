# Getting Started

LiteQuote for WooCommerce replaces the "Add to Cart" button with a "Request a Quote" button on selected products. Visitors can request a quote in under 10 seconds without leaving the product page.

## How It Works

1. **You mark a product** as "Price on request" (checkbox or zero price)
2. **The visitor sees** a quote button instead of the purchase button
3. **They click it** and a modal popup appears with a pre-filled form
4. **They submit** their name, email, phone, company, quantity, and message
5. **You receive** a professional HTML email with all the details + PDF attachment
6. **You open the dashboard**, type a price, and click "Send Quote"
7. **The customer receives** a professional PDF quote with price table

## System Requirements

| Requirement | Minimum |
|---|---|
| WordPress | 6.6+ |
| WooCommerce | 8.0+ |
| PHP | 8.0, 8.1, 8.2, 8.3 |
| SMTP Plugin | FluentSMTP (free, recommended) or WP Mail SMTP |

::: warning SMTP Required
WordPress does not send emails reliably without an SMTP plugin. Install [FluentSMTP](https://wordpress.org/plugins/fluent-smtp/) (free) **before** configuring LiteQuote. See [SMTP Setup](/guide/smtp-setup) for step-by-step instructions.
:::
